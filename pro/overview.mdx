Retype Pro and Enterprise unlock additional features within the same `retypeapp` installation you already have. No separate product or installer is required — adding your key activates everything automatically.

<Note>

</Note>

## Try Pro before purchasing

You can preview all Pro features locally without a key.

**Option 1:** Pass the `--pro` flag when starting:

```bash
retype start --pro
```

**Option 2:** Add `pro: true` to your `retype.yml`:

```yaml retype.yml
start:
  pro: true
```

<Note>
  In preview mode, the "Powered by Retype" branding remains enabled and the 100-page limit is still enforced. All other Pro features are available for testing.
</Note>

## Activate your key

After purchasing, your key is emailed to you. See the [key setup guide](/guides/key-setup) for full instructions on the three ways to activate it:

- Add to your local wallet with `retype wallet --add <key>`
- Pass per-build with `retype build --key <key>`
- Set the `RETYPE_KEY` environment variable (recommended for CI/hosting)

## Licensing details

<AccordionGroup>
  <Accordion title="Does my website stop working if I don't renew?">
    No. Your project continues to build and serve pages indefinitely, even without renewal. The only limitation is that you cannot upgrade to a newer version of Retype released after your key's end date.
  </Accordion>
  <Accordion title="How many users or developers are covered?">
    Licenses are per project (website), not per person or server. One license covers an unlimited number of users, authors, and developers working on that project.
  </Accordion>
  <Accordion title="What counts as a project?">
    Each `retype.yml` file used to generate a website is one project. The `url` setting in your `retype.yml` must match the URL registered to your key.
  </Accordion>
  <Accordion title="Can I change the URL registered to my key?">
    Within the first 15 days of purchase, one change to a different domain or subdomain is allowed. Send a request to hello@retype.com. After 15 days, a new key is required.
  </Accordion>
  <Accordion title="Can I purchase multiple licenses?">
    Yes. Each license is created and managed separately. Purchase one now and additional licenses at any future time.
  </Accordion>
  <Accordion title="Are custom licensing options available?">
    Yes. Email hello@retype.com with your requirements.
  </Accordion>
</AccordionGroup>
